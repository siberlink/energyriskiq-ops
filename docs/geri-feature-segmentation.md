# GERI — User Dashboard Feature Segmentation

Optimized for existing data assets: GERI + Brent + TTF + VIX + EUR/USD + EU Gas Storage

---

## Free User Plan (Dashboard Access Level 1)

**Goal:**
- Show value
- Encourage upgrade
- No analytical edge

### Dashboard Panels

#### 1. GERI Snapshot Panel

User sees:
- Today's GERI value
- Risk Band label
- Arrow trend (Up / Down / Stable)

#### 2. Mini Trend Panel

- 7-day sparkline only
- No zoom
- No historical navigation

#### 3. Simplified Drivers Panel

Shows top 3 macro categories only:
- Geopolitical
- Energy Supply
- Market Stress

**Not Included:**
- No region
- No severity scoring
- No event context

### Charts Module

#### GERI History Chart

**Limit:** Last 14 days only

#### Overlay Capability

- ONE preset chart only: **GERI vs Brent** (fixed)
- No switching assets
- No toggles
- No smoothing
- No controls

### Interpretation Panel

Auto-generated 2–3 sentence explanation of current risk level.
